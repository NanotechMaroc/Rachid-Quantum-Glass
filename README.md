🌟 الرؤية (The Vision)
مشروع مغربي طموح يهدف إلى تحويل واجهات ناطحات السحاب إلى محطات طاقة نظيفة دون المساس بالشفافية، باستخدام خوارزمية هندسية متطورة .
🛠 التقنيات المستخدمة (Technology Stack)
• Quantum Dots (CIGS/ZnS): النقاط الكمومية لاصطياد الأشعة تحت الحمراء.
• Nanotechnology: طلاء نانوي لرفع كفاءة الانكسار الضوئي.
• Physics Engines: محاكاة المسارات الضوئية عبر (Monte Carlo Method).
