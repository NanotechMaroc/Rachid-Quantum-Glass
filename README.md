 الرؤية (The Vision)
مشروع مغربي طموح يهدف إلى تحويل واجهات ناطحات السحاب إلى محطات طاقة نظيفة دون المساس بالشفافية، باستخدام خوارزمية هندسية متطورة .
🛠 التقنيات المستخدمة (Technology Stack)
• Quantum Dots (CIGS/ZnS): النقاط الكمومية لاصطياد الأشعة تحت الحمراء.
• Nanotechnology: طلاء نانوي لرفع كفاءة الانكسار الضوئي.
• Physics Engines: محاكاة المسارات الضوئية عبر (Monte Carlo Method).
 خارطة الطريق (Roadmap)
• [ ] Phase 1: كتابة الكود الرياضي للمحاكاة (Python).
• [ ] Phase 2: تحديد معاملات الانكسار الضوئي (Optical Indexing).
• [ ] Phase 3: البحث عن شركاء لتصنيع النموذج الأولي (Prototyping).
نتائج المحاكاة الأولية (Simulation Results)
بناءً على المحاكاة البرمجية لعام 2026 في مدينة ورزازات، أظهرت النتائج ما يلي:
• كفاءة امتصاص الفوتونات: 70% بفضل النقاط الكمومية.
• القدرة الإنتاجية: نافذة واحدة (2م²) يمكنها إنتاج حوالي 4.5 كيلوواط ساعة يومياً.
• التطبيق: ناطحة سحاب بـ 1000 نافذة يمكنها توليد 4.5 ميجاواط ساعة يومياً، مما يكفي لتشغيل المبنى بالكامل وتصدير الفائض للشبكة.
ملاحظة: تم حساب هذه النتائج باستخدام محرك الفيزياء الخاص بالمشروع physics_engine.py.
